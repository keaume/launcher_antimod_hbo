# 🏨 Habbo Launcher Antimod

Projet Node.js avec Express et PostgreSQL, Railway, bcrypt.

Une fois connecté, et le code d'identification de la connexion Habbo rempli, l'application va permettre de déployer une couche supplémentaire afin d'éviter la détection de modération (les dés vont alors tourner client-side de la seconde couche).

Interface inspirée du style rétro Habbo.

## Stack

- Node.js
- Express
- PostgreSQL
- bcrypt
- express-session
